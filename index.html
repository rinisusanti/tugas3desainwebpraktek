Dashboard

Tampilan Dashboard yang user friendly dan mudah dipahami. Dashboard memberikan informasi rekapitulasi hari ini, . Laporan dalam dashboard adalah laporan singkat kondisi di rumah kecantikan, laporan detail dapat diperoleh dalam setiap modul.

Rekap

Kemudahan mengelola data di rumah kecantikan. Data ini bisa memberikan informasi siapa saja yang telah melakukan konsultasi.

Transaksi Pembayaran

transaksi keuangan ini untuk mencatat daftar member yang masuk, memberr ini berasal dari pembayaran. Transaksi ini memberikan kemudaha pengaturan terhadap pembayaran.

Impor Data Dari XLS

memberikan kemudahan dalam proses input data dengan menyediakan fasilitas import data dari excel/spreadsheet. Fasilitas ini mempercepat dan mempermudah dalam akses informasi admin

Presensi Pegawai Per Hari

mempermudah admin dalam mendata Presensi pegawai. Kadang-kadang ada Pegawai yang hadir di pagi hari tapi siang tidak hadir. Maka Pasti bisa dapat digunakan untuk melakukan presensi pegawai setiap jam.

Jadwal Kerja

Admin dapat melihat jadwal kerjanya setiap hari, mempermudah admin dalam mengaksesnya.

Daftar Laporan

Laporan dapat langsung diakses oleh pegawai, sehingga admin harus selalu update terhadap hasil kerja.
